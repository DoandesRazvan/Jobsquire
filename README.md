Welcome to Jobsquire!

Jobsquire is an app that uses web scrapers and an AI model to search for and filter job postings based on your criteria.

The app works by having a search field where you can input your desired search parameters and select from five of the most popular job-searching websites in Romania, enabling you to search for your desired role on multiple sites at once, improving speed.

The results will be served up on this same page and you will then be able to interact with each one as well as filter the results based on specific criteria (If the results are relevant to the role you're looking for or by WFO/WFH/Remote positions). You can also download the results locally in case you don't want to use the Jobsquire filters.

*Technical info:Jobsquire uses web scrapers to search for job postings and uses an integrated Gemini 3.1 Flash Lite AI model to filter the job results.
AI might make mistakes so if after filtering the results you feel like a mistake was made, you can restore the initial results and download the results file for your own filtering, and you'll also have access to the search link for every result in case you want to do your own manual search.

This application is open source and was designed and created by Doandes Razvan.
